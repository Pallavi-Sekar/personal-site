# Pallavi Sekar | Software Developer Portfolio

This is a personal portfolio website for **Pallavi Sekar**, showcasing skills, work experience, projects, and contact information. The website is designed with a modern, responsive layout, smooth scrolling, and interactive carousels for skills, experience, and projects.

**Live Site**: [https://pallavisekar.netlify.app/](https://pallavisekar.netlify.app/)

---

## Features

- **Responsive Design**: Works on desktop and mobile devices.
- **Smooth Scrolling**: Navigating through sections is seamless.
- **Sections Included**:
  - About Me
  - Skills (with interactive carousel)
  - Work Experience (carousel format)
  - Projects (with links to GitHub)
  - Contact Information (email, phone, social media links)
- **Downloadable Resume**: Users can download the resume directly from the website.
- **Interactive UI Elements**: Buttons, cards, and social media icons with hover effects.
- **Modern Styling**: Glassmorphism-inspired backgrounds with blur effects.

---

## Technologies Used

- **Frontend**: HTML5, CSS3, JavaScript
- **Styling**: Custom CSS with glassmorphism and hover animations
- **Assets**: Images, icons, and downloadable PDF resume
- **Frameworks/Libraries**: None (pure HTML/CSS/JS)
